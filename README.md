# 小红书图片生成器

这是一个用于生成“小红书转折型封面+内页”的工具

一个独立的小红书图片生成工具，保留 brand-next 的奶油色高级感风格。

## 功能特性

- 🎨 **品牌风格**：保留 brand-next 的奶油色背景和深咖色字体风格
- 📱 **标准尺寸**：生成 1242×1656px 的小红书标准尺寸图片
- 🖼️ **封面生成**：支持主标题、副标题，可上传背景图片
- 📄 **内页生成**：支持 Markdown 风格标记（# 一级标题、## 二级标题），可插入多张图片
- 🎯 **图片插入**：内页模式下可上传多张图片，并指定插入位置
- 🌈 **背景图片**：封面和内页均支持自定义背景图片

## 技术栈

- **Next.js 15** - React 框架
- **TypeScript** - 类型安全
- **node-canvas** - 服务端图片生成
- **CSS Variables** - 品牌样式系统

## 安装

```bash
# 安装依赖
npm install
```

## 运行

```bash
# 开发模式
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm start
```

访问 http://localhost:3000 即可使用。

## 使用说明

### 封面模式

1. 选择"封面"类型
2. 填写主标题（必填）
3. 填写副标题（可选）
4. 可上传背景图片（可选）
5. 点击"生成图片并下载"

### 内页模式

1. 选择"内页"类型
2. 填写主标题（可选）
3. 填写副标题（可选）
4. 填写正文内容，支持 Markdown 风格：
   - `# 一级标题` - 52px，字重 600
   - `## 二级标题` - 44px，字重 500
   - 普通文本 - 36px，字重 400，行距 1.6
5. 可上传背景图片（可选）
6. 可上传多张内容图片，并指定插入位置（可选）
7. 点击"生成图片并下载"

## 字体体系

### 封面字体

- **主标题**：125px，字重 600，深咖啡色 (#2f251f)
- **副标题**：60px，字重 400，深咖啡色 (#2f251f)

### 内页字体

- **一级标题**：52px，字重 600，色号 #3A2F2C
- **二级标题**：44px，字重 500，色号 #4B3F3C
- **正文**：36px，字重 400，色号 #4B3F3C，行距 1.6

## 注意事项

- 需要 Node.js 18+ 版本

- 生成的图片为 PNG 格式，尺寸为 1242×1656px
